O tradutor traduz códigos de arquivos de texto .blp para assembly.
Esses códigos devem estar na linguagem descrita no PDF (Trabalho-v1.pdf).
Há mais informações sobre o projeto nesse mesmo PDF.
Para executar o código, basta compilar o arquivo tradutor.c para um executável run, e, no terminal, executar:

Windows:

run < arquivoDeTexto.blp

Linux:

./run < arquivoDeTexto.blp
